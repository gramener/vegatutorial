Changing chart types -- scatterplot

This example shows how to plot scatterplot based on barchart example.

The Vega spec scatter.json is quite similar to how we constructed the barchart. Let's look at what's different.

The "scales" section defines how to map data values to visual values (pixel positions, colors etc.). Here, of two scales, X "xscale" needs few modifications. 

    "scales": [
      {
        "name": "xscale",              // Create a scale named "xscale"
        "type": "linear",              // on linear scale, and not ordinal
        "domain": {"data": "table",    //
                   "field": "x"},      //
        "range": "width",              //
      },

The "marks" type is defined as "symbol"

    "marks": [
      {
        "type": "symbol",                 // mark type is symbol

The "properties" have similar definitions as barchart. However, in scatterplot "width" under ("enter") needn't be defined and "size"is assigned to symbols

    "enter": {                                 // Set these properties for symbols
      "x": {"scale": "xscale", "field": "x"},  // x value by applying "xscale" to "x" data
      "y": {"scale": "yscale", "field": "y"},  // y value by applying "yscale" to "y" data
      "fill": {"value": "steelblue"},          // filled with blue
      "size": {"value": 100}                   // pixel area of the symbol
    }
